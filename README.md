# New Spirituality Calendar 修行日历订阅链接


一个面向修行、正念、自我觉察与灵性生活实践者的开源日历项目。  
本项目将分散在不同传统、网页或文化脉络中的修行相关日期整理为可订阅的数字日历，方便用户在日常生活中通过 iPhone、iPad、Mac、Outlook 及其他支持 iCalendar 的日历软件进行查看和提醒。

## 项目简介

这个项目最初源于我个人的真实需求。  
在持续进行冥想、正念、佛教修持与相关灵性实践的过程中，我希望能有一种更方便的方式，把重要的修行日期整合到日常生活使用的电子日历中，而不是每次都临时去查网页、截图保存，或依赖零散的信息来源。

因此，我开发并整理了这一组可订阅的修行日历，希望它既能服务我自己的日常实践，也能为其他对精神成长、灵性修行、自我观察、斋戒日或宇宙节律感兴趣的人提供一个长期、稳定、免费、可持续更新的工具。

## 项目包含的日历

本项目目前包含以下几类日历：

- 佛教六斋日
- 佛教十斋日
- 北京版断食日历（基于北京地区时间/页面信息整理）
- 重要宇宙天象事件日历
- 印度 Ekadashi 断食日日历
- 综合版修行日历

其中，**综合版修行日历**会汇总多个主题日历中的事件，并在标题前添加类别前缀，帮助用户快速区分不同类型的修行提醒。

## 每个日历的用途

### 1. 佛教六斋日
适合希望在每月固定日期进行持戒、素食、反省、自我观察或加强修行的人使用。

### 2. 佛教十斋日
适合希望更系统地按照传统佛教修行节律安排生活的人使用，也适合作为个人提醒工具。

### 3. 北京版断食日历
此部分根据北京地区相关断食日信息进行整理，适合希望结合本地时间参考断食安排的人使用。

### 4. 宇宙天象事件日历
收录重要天象节点，适合对节气、天文变化、仪式感生活、周期性自我回顾与内在觉察感兴趣的使用者。

### 5. Ekadashi 断食日日历
Ekadashi 是印度传统中重要的断食与净化日。  
该日历适合对印度灵修传统、瑜伽文化、身心净化实践或有规律断食感兴趣的人使用。

### 6. 综合版修行日历
如果你不想分别订阅多个日历，可以直接使用综合版。  
它会尽可能整合不同来源的重要修行日期，方便统一管理，也更适合长期使用。

## 如何订阅

你可以将对应日历链接添加到支持 iCalendar（`.ics`）订阅的日历软件中，例如：

- Apple Calendar（iPhone / iPad / Mac）
- Outlook
- Google Calendar（部分情况下可通过 URL 导入或订阅）
- 其他支持 iCal / ICS 的日历客户端

### 一般订阅步骤

1. 找到你想使用的日历订阅链接
2. 复制该链接
3. 打开你的日历软件
4. 选择“通过 URL 订阅”或“添加互联网日历”
5. 粘贴链接并保存
6. 等待日历同步完成

> 不同设备和软件的界面略有不同，但原理基本一致。

## 项目特点

- **免费订阅**：任何人都可以使用
- **自动更新**：适合长期订阅，不需要重复手动添加
- **跨设备同步**：支持在手机、平板、电脑等设备查看
- **分类清晰**：不同主题可单独订阅，也可使用综合版
- **开源可修改**：你可以基于自己的需要调整内容或继续开发
- **从真实需求出发**：不是抽象概念项目，而是源于长期实践中的真实使用需求

## 技术实现

本项目使用 Python 生成日历文件，并通过 GitHub Pages / 仓库内容进行管理和发布。  
项目结构中包含配置文件、生成脚本和文档页面，便于后续维护、增补与迭代。

如果你希望在此基础上继续开发，你可以：

- 修改配置文件中的事件来源
- 增加新的修行主题日历
- 优化事件命名方式和分类前缀
- 调整更新频率
- 重新发布你自己的订阅版本

## 关于综合版

综合版会汇总多个主题日历中的事件，并尽量在事件标题中增加类别标记，方便识别。  
如果你希望只保留一个修行日历入口，那么综合版会是最方便的选择。
